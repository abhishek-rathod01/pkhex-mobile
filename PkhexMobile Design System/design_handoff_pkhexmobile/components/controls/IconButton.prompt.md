Icon-only square button for toolbars, nav bars, and inline affordances. Always pass `label` for a11y.

```jsx
<IconButton label="Back" onClick={goBack}><ChevronLeft/></IconButton>
<IconButton variant="primary" label="Edit"><PencilIcon/></IconButton>
```

Variants: ghost (default), outline, primary. Sizes sm(34)/md(44)/lg(52).
